# File Conventions

| Type | Pattern | Example |
|------|---------|---------|
| Client work | `clients/[client-name]/` | `clients/acme-corp/` |
| Active campaigns | `campaigns/[campaign-name]/` | `campaigns/fintech-q2/` |
| Skills | `.claude/skills/[name]/SKILL.md` | `.claude/skills/cold-email/SKILL.md` |
| Company docs | `company/[doc].md` | `company/ICP.md` |
| Archive | `archive/[type]/` | `archive/campaigns/` |

Never save to workspace root. Archive, don't delete.
